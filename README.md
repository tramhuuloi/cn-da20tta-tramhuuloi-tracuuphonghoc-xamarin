## Hướng Dẫn Cài Đặt

1. Clone dự án từ repository:

    ```bash
    git clone https://github.com/your-username/your-project.git
    ```

2. Di chuyển vào thư mục dự án:

    ```bash
    cd your-project
    ```

3. Cài đặt các dependencies:

    ```bash
    npm install
    ```

## Hướng Dẫn Sử Dụng

1. Chạy ứng dụng:

    ```bash
    npm start
    ```

2. Truy cập ứng dụng từ trình duyệt:

    ```
    http://localhost:3000
    ```
